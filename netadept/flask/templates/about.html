Welcome to the netAdept Automation platform



netAdept aims to be the singlepoint where from to monitor and manage all network devices



About netAdept       Technologies used       Contact Information      



netAdept Features : 




1. Device Inventory
	a. Add devices to access.
	b. Assign features to each device (Credentials, SSH ports, IP addresses.
	c. Create groups allowing filtering devices to select and features to assign to each device group.

2. Show Commands
	a. Access infromation on each device agnostic of vendor.
	b. Download output to local device.
	c. urrently Cisco IOS, Juniper junos and Huawei devices are currently supported (to be expanded as per demand). 

3. Backups
	a. Automate and schedule device backups.
	b. save backup files to the netAdept server, and delete or download as required.
	
4. File Manager
	a. Upload and download files from/to PC.
	b. Push files to devices for upgrades.
	
5. User Management
	a. Basic user management by authorising and removing email addresses.
	
6. Basic Troubleshooting
	a. Ping and Tracroute. 
	b. Dashboard showing which Inventory devices are reachable.
	
7. Resources
	a. Users, Authed emails and Inventory is saved in MongoDB. 
	b. Inventory devices are automatically added to Zabbix Monitoring platform collecting SNMP data.
	c. Build network diagrams using DrawIO.
	
8. Miscellaneous Tools
	a. DNS and IP address lookups
	b. Password Generator
	c. Subnet Calculator
