Welcome to the netAdept Automation platform



netAdept aims to be the singlepoint where from to monitor and manage all network devices



About netAdept       Technologies used       Contact Information      



netAdept Features :



  • Create an inventory of network devices.

  • Create device groups allowing advanced filtering and selection of multiple devices to connect to.

  • Send show commands any devices agnostic of vendor and retrieve that output. Currently Cisco IOS, Juniper junos and Huawei devices are currently supported (to be expanded as per demand).

  • Automate and schedule device backups and save backup files to the netAdept server for easy access.

  • Manage files. Upload and download files from/to PC, netAdept and network devices.

  • Manage user users that are allowed to create accounts and connect to netAdept.

  • miscellaneous tools like domain and ip address lookups, subnet calculator and password generator.

  • Zabbix monitoring suite is used to automatically add devices in the netAdept inventory to Zabbix.
