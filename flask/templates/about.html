Welcome to the netAdept Automation platform



netAdept is a python based web UI that uses various open source technologies to simplify connecting to network devices and pull information from these devices. The aim is to be agnostic of all network device vendors and draw out any required information in the most direct way in order to enable network engineers to correct any network issues as quickly as possible.



Technologies used:



Nornir - Network automation framework

netmiko - Python based SSH library

Scrapli - Python based SSH library

Zabbix - monitoring suite

MongoDB - database

GoTTY - web base command line tool


DrawIO - Diagramming tool
